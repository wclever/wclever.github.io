mac下搭建GitHub+Hexo博客

发表于 2020-10-15 更新于 2020-10-16 分类于 总结

注册一个GitHub账号

直接在github.com，操作注册。

本地环境搭建

使用Hexo博客框架需要Node.js环境，查看是否安装环境

node -v
npm -v
git --version
hexo -v

如果没有安装环境，则先安装环境

Node.js
1.官网下载安装包安装Node.js官网
2.命令行 brew install node

Git
brew install git

Hexo
sudo npm install -g hexo-cli

博客初始化

创建存储博客的文件夹，比如在~/blog
mkdir blog

进入blog文件夹下
cd blog

执行以下命令初始化本地博客，并会下载一系列文件
hexo init

执行以下命令安装npm
sudo npm install

以上安装完成后，即可使用

生成本地静态网页缓存
hexo g或者hexo generate

本地服务器运行测试
hexo s或者hexo server
打开浏览器，在网址中输入http://localhost:4000，有网页显示就证明成功了

本地博客关联GitHub

注册并且登录GitHub后，点击右上角‘+’中Create a new repository，创建一个新的仓库，注意：名字格式必须为username.github.io，比如我的GitHub用户名为wclever，那么我创建的仓库名就为wclever.github.io，选择下面生成READNE.md文件，然后创建。

目前博客仅能在本机查看，如果想发布在GitHub上需要关联GitHub账号
