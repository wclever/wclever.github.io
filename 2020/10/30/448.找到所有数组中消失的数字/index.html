448. 找到所有数组中消失的数字

给定一个范围在 1 ≤ a[i] ≤ n ( n = 数组大小 ) 的 整型数组，数组中的元素一些出现了两次，另一些只出现一次。

找到所有在 [1, n] 范围之间没有出现在数组中的数字。

您能在不使用额外空间且时间复杂度为*O(n)*的情况下完成这个任务吗? 你可以假定返回的数组不算在额外空间内。

示例:

思路

两遍循环，第一遍将数值找到对应的索引，然后对索引上的数+n，第二遍循环找，如果数值不存在的数组中的值是小于n的

时间复杂度：$\mathcal{O}(N)$

空间复杂度：$\mathcal{O}(1)$
