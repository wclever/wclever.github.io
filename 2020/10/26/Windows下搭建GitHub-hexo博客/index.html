Windows下搭建GitHub+hexo博客

发表于 2020-10-26 分类于 总结

安装Node.js和Git

Node.js官网：nodejs.org/zh-cn/
Git官网：git-scm.com/
注意：最好都安装在系统盘下，默认为C盘，安装全部默认就行

使用cmd查看node、npm、Git版本号
node -v
npm -v
git --version

注册GitHub账号以及创建代码库

创建账号后，进入个人主页，点击右上角的加号“+”，选择New repository

注意，代码库的名称一定是yourname.github.io，否则后面开启gh-pages后不能访问yourname.github.io，yourname是自己的GitHub用户名，在此勾选下面自动生成README.md文件，因为hexo部署不能是空的内容。

开启gh-pages功能

点击界面右侧的Settings，则打开这个库的setting页面，向下拖动，看见GitHub pages

本地配置hexo

在本地合适位置创建存放博客的文件夹，比如存放位置为C:\document\blog，然后在文件夹空白处按住shift+鼠标右键，点此打开命令行窗口，执行命令

输入hexo d，出现对话框，输入yes回车，出现提示RSA未添加的报错，说明没有配置ssh

配置ssh

在GitHub个人主页右上角头像->settings->SSH and GPG keys

点击New SSH

git bash中输入，生成本机ssh key

引号里面的内容输入你的邮箱地址，然后回车，会提示你文件保存的路径，这时候按回车键确认，然后会提示你输入密码，输入即可（输入密码是看不到的），然后会确认输入一次，就可以在刚刚的路径看到生成了两个文件，一个是id_rsa，另一个是id_rsa.pub，用sublime打开id_rsa.pub然后选中里面的全部内容，复制下来。

这里的Title随便填写，主要是为了方便管理密钥，然后把刚刚拷贝的内容粘贴到Key里面去，点击Add SSH key
到此，Github上面的SSH配置就算完成了
